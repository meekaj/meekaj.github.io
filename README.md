# Mihika Jain's Portfolio

Welcome to the GitHub repository for my personal portfolio website! This site showcases my projects, skills, and professional details.(Currently WIP!)

## Overview

This portfolio is designed to present my educational background in Economics, programming skills, and projects I have worked on. It also dynamically updates to display my latest projects hosted on GitHub.

## Features

- **Dynamic Project Listings**: Automatically fetches project details from my GitHub repositories and displays them.
- **Responsive Design**: Ensures a good viewing experience on devices of various sizes.
- **Interactive Elements**: Includes interactive elements such as links to project repositories and my LinkedIn profile.

## Technologies Used

- HTML
- CSS
- JavaScript
- [GitHub API](https://docs.github.com/en/rest)
